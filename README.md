## ❓ Frequently Asked Questions

**Does the software delete files?**
No. This tool only sends simulated emails to track click rates. It does not look at your personal files or change your system settings.

**Can I run the software on a Mac?**
This version is designed for Windows. Please check back for updates regarding other platforms.

**What happens if I click a link in a simulation?**
You will be directed to a training page. This page explains what suspicious elements were inside the email. No harm comes to your computer.

**How do I update the software?**
The software checks for new versions when you open it. If a new version exists, the app will ask for permission to update. Accept the update to get the latest security improvements.

**Is my data secure?**
All drill data stays on your local machine. We do not look at your reports or your employee information.

## 📖 Troubleshooting

If you have trouble running the software, check the list below.

- Software does not open: Restart your computer and try again. Sometimes background processes block new apps.
- No emails sent: Check your internet connection. Ensure you configured your API gateway settings correctly.
- Application error: See if you have the latest version of Windows installed. Updates often resolve compatibility issues.
- Missing reports: Go to the "History" tab to view past drills. If your screen is blank, ensure you clicked "Save" after you ran the test.
